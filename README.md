# 🌌 Interactive Faction Selector 🌌
Welcome to my project! This is an immersive and dynamic faction selector inspired by the Bleach universe, built using HTML and CSS. The goal of this project is to highlight user interactivity, creative styling, and responsive design principles.

## Features
- **Dynamic Background Design**
```
- The webpage uses a visually engaging GIF background (`Bleachgif.gif`) to immerse users.
- Smooth transitions are applied when users interact with buttons, changing the background seamlessly.
```

- **Faction-Specific Buttons**
```
- Includes two buttons that represent distinct factions:
  - **Soul Society**: Styled with bold red text on a black background, inspired by a Japanese aesthetic.
  - **Wandenreich**: Styled with clean white text on an aqua background, reflecting a Gothic vibe.
- Buttons are interactive:
  - Hover Effects: Color and background adapt when hovered.
  - Transitions: Smooth visual effects for hover interactions.
```

- **Typography**
```
- Custom Google Fonts are integrated for distinct styling:
  - **Orbitron**: Adds a futuristic and tech-like feel to the page.
  - **Noto Sans JP**: Reflects the traditional Japanese theme for Soul Society.
  - **Germania One**: Emphasizes a Gothic aesthetic for Wandenreich.
```

- **Visual Focus**
```
- The logo (`Bleachlogo.png`) is prominently displayed at the top, setting the theme.
- The text "Who are you?" invites users to choose their faction, creating an engaging and thematic experience.
```

## Key HTML Elements Used 
- **Semantic Tags**
```
- <h1>: Used for the primary question to create a clear hierarchy.
- <button>: Represents the faction selection options with accessible design.
- <img>: Displays the logo and provides a visual identity to the webpage.
```

- **Attributes**
```
- class: Used for styling buttons with specific faction-related properties.
- id: Applied to unique elements such as the logo and dynamic text for styling.
```

## Key CSS Elements Used 
- **Transitions**
```
- Smooth animations are applied to buttons and the background to enhance user experience.
- Example: `transition: background 0.3s ease, color 0.3s ease;`
```

- **Custom Fonts**
```
- Google Fonts integration ensures a unique and consistent aesthetic:
  - Orbitron: Futuristic and tech-inspired.
  - Noto Sans JP: Japanese-inspired.
  - Germania One: Gothic and thematic.
```

- **Opacity**
```
- Buttons have a subtle transparency (`85%`) to create a polished and modern look.
```

- **Responsive Design**
```
- Button padding and margins ensure proper alignment and spacing across different screen sizes.
```
